Projects with Ooredoo

Below are the projects that I've worked during in Ooredoo

Loyalty Module
Python, Flask, MySQL, RedHat Linux, microservices

Earning, Burning, and User are the three primary modules. developed using a microservice architecture and load-balanced using two nodes. Every service is redundant, and websocket architecture is used to transfer data between services. The majority of critical calculations and configurations are divded up into config snippets which greatly improves maintenance and scalability.

Out Bound Dialing
Asterisk, PBX, Python, Flask, HTML, CSS, bootstrap

Asterisk is set up with SIP trunk to the core servers. After that, Python is used to control the backend of the OBD, such as charging, call throttling, routing with Asterisk python Library. Alongside with customer portal and admin portal, customer portal for the users to perform batch call using OBD services and to check statistics. Admin Portal is for the interal utilization where admins can insert new users, bind packages, make new caller-ids.

E-Sim
Solution architecture and design thinking

E-sim has two main modules, upgrading to esim and new sim sale. I contributed to brainstorming sessions for both of those modules. Upgrading to Esim is about swapping the physical sim to digital sim. And New sim sale is about buying new sim via My Ooredoo App or web portal.

VOLTE
Python, Pydantic, SOAP, core servers

OLTE (Over LTE) enables users to make voice calls over Long Term Evolution (LTE) networks, where traditional calls typically utilize 2G/3G networks. With VOLTE, voice traffic is routed through 4G networks, enhancing call quality and reliability. I kicked off this by manually sending specific soap commands with python to core servers to make sure every single command works.
